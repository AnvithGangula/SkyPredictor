### üöÄ **About The Project**

**SkyPredictor** is a web-based weather application that provides accurate and timely weather forecasts. It was founded in 2024 by a team of passionate meteorologists and technology enthusiasts. The goal was to combine advanced meteorological science with cutting-edge technology to deliver forecasts that are not only precise but also easy to understand.

**Key technologies and libraries used in this project include:**
* **HTML:** Provides the structure for all web pages, including the homepage, about, and weekly forecast.
* **CSS:** Used for styling the entire application, including custom fonts like `Jost`, `DM Sans`, and `Poppins`, and a responsive design approach.
* **JavaScript:** Powers interactive elements such as the dynamic contact card, form validation for registration, and dynamic weather data display.
* **Bootstrap:** Used for the layout and styling on the weekly weather forecast page.
* **Font Awesome:** Provides the icons for buttons on the weather page.
* **Axios:** A JavaScript library for making HTTP requests, which is likely used to fetch weather data from an API.

### üèÅ **Getting Started**

This project is a static website and doesn't require a complex setup. You can run it directly in a web browser.

#### **Prerequisites**

You only need a modern web browser like Google Chrome, Mozilla Firefox, or Microsoft Edge.

#### **Installation**

1.  Clone the repository to your local machine:
    ```sh
    git clone https://github.com/AnvithGangula/SkyPredictor
    ```
2.  Navigate to the project directory:
    ```sh
    cd SkyPredictor
    ```
3.  Open the `HomePage.html` file in your preferred web browser to view the application.

### üíª **Usage**

* **Homepage (`HomePage.html`):** The main entry point, featuring a navigation bar to other pages.
* **Weekly Forecast (`index.html`):** The main weather page where users can enter a city to get a forecast.
* **Registration (`Register.html`):** A form for new users to create an account. The page validates the username, email, and password.
* **Login (`Login.html`):** A form for existing users to log in to the application.
* **About Us (`About.html`):** Provides a detailed description of the project's mission and team.
* **Contact (`Contact.html`):** Features an interactive contact card.
